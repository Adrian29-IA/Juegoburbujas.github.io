Instrucciones del Juego

Objetivo

  • Atrapa las burbujas buenas haciendo clic sobre ellas.
  • Evita hacer clic sobre las burbujas malas.
  • Ganas puntos por cada burbuja buena atrapada.
  • El juego termina si se acaban las vidas o se agota el tiempo.

Burbujas Buenas

  • Son azules y transparentes.
  • Al hacer clic: +1 punto, se reinicia el tiempo y aparece otra.

Burbujas Malas

  • Son rojas y transparentes.
  • Al hacer clic: -1 vida, se reemplaza con otra burbuja mala.

Vidas

  • Empiezas con 3 vidas.
  • Si llegas a 0, el juego termina.

Tiempo

  • Tienes 3 segundos para atrapar una burbuja buena.
  • Si no lo haces a tiempo, pierdes automáticamente.

Controles

  • Mouse: Clic para atrapar burbujas.
  • Teclado: Presiona R para reiniciar después de perder.

Fin del Juego

  • Se muestra el mensaje "¡Juego terminado!" y tu puntaje final.
  • Puedes reiniciar presionando R.
